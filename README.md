# 🎓 Local AI Study Buddy

An AI-powered local study assistant built using **Streamlit**, **Ollama**, and **Gemma3**.
This application helps students learn topics, generate quizzes, and study privately using local Large Language Models (LLMs).

---

# 🚀 Features

* 📚 AI-powered topic explanations
* 🧠 Quiz generation with answers
* 🎯 Education-level customization
* 🖥️ Fully local AI using Ollama
* 🔒 100% private (no cloud processing)
* ⚡ Real-time streaming responses
* 🎨 Clean and modern Streamlit UI
* 🤖 Supports multiple LLMs

---

# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/local-ai-study-buddy.git

cd local-ai-study-buddy
```

---

## 2️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

# 🧠 Install Ollama

Download and install Ollama:

https://ollama.com

---

# 📥 Pull AI Model

Recommended model:

```bash
ollama pull gemma3
```

Other supported models:

```bash
ollama pull llama3

ollama pull mistral

ollama pull deepseek-coder
```

---

# ▶️ Run Ollama Server

```bash
ollama serve
```

---

# ▶️ Run Streamlit App

```bash
streamlit run app.py
```

# 🎯 How It Works

1. User selects:

   * Education level
   * Subject
   * Learning mode

2. The app sends prompts to the local LLM using Ollama.

3. AI generates:

   * Structured explanations
   * Quiz questions
   * Answers with reasoning

4. Responses are streamed live in the UI.
